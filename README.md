# Pamela Kaye Magnifico | Personal Portfolio

A minimalist, responsive portfolio website designed to showcase my projects in **Cybersecurity**, **Data Science**, and **Hardware Engineering**.

Built with a focus on clean UI, performance, and the "Deep Navy & Cyan" aesthetic.

## Key Features
* **Deep Navy Theme:** Custom linear gradient background (`160deg`) with neon cyan accents.
* **Responsive Design:** Fully fluid layout that adapts from desktop to mobile.
* **Interactive UI:**
    * **Tabbed Experience Section:** Toggle between different work experiences without page reloads.
    * **Project Filtering:** Sort projects by "Software" or "Hardware" categories.
    * **Custom Contact Modal:** A sleek, glass-morphism overlay form.
* **Hardware Integration:** Specialized section showcasing PCB designs and engineering documentation.

## Tech Stack
* **Frontend:** HTML5, CSS3 (Custom Variables, Flexbox, Grid), JavaScript (ES6+)
* **Icons:** FontAwesome 6.4
* **Fonts:** Fira Code (Monospace) & Inter (Sans-serif)
